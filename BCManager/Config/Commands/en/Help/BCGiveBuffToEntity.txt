    Aliases: {commands}
    Default Options: {defaultoptions}
    Usage:
      1. {command}
      2. {command} list
      3. {command} <entityid>
      4. {command} <entityid> <buffid>
      5. {command} <EntityClass> <buffid> /type
    Notes:
      Will display all entities and what buffs they have, or provide an entity id to view just that entity
      Use bc-help to get a list of common /options
      Use bc-go buffs to get a list of buff ids
      Use bc-le to get a list of spawned entity types

      bc-geb list - will display all entities and the buffs they currently have
      bc-geb <entityid> - will display the buffs for just that entity
      bc-geb <entityid> <buffid> - will add the specified buff to the entity
      <EntityClass> <buffid> /type - will add the specified buff to all entities of the specified entity class
