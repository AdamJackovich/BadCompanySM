List Players Extensive Data
      Aliases: bc-lpx, bc-lx
      Usage:
            1. bc-lpx <partialname / entityid / steamid>
            2. bc-lpx
      Notes:
            Command without a param will return all datasets. Be aware this is an extensive list on large servers and may cause server load. Use sparingly.
            <partialname> case insensitive. Will search online players only. If more than one player matches will return an error to console, refine your text if this happens.
            <entityid> Will search online players only. Use lp to get a list of player entityids
            <steamid> Can search for offline players.
